# Learning-Augmented Online Acknowledgment with Worst-Case Latency Guarantees

This repository contains implementations and benchmarking code for online algorithms that solve the TCP acknowledgment problem with and without predictions. 

## Requirements

- Python 3.7 or newer: Download from [Python's Website](https://www.python.org/downloads/).
- Pip: Download from [PyPA's website](https://pip.pypa.io/en/stable/installation/).
- Install the following packages.

```bash
pip install numpy matplotlib dpkt
```

## Usage
From the directory containing `acknowledgment.py`, run:
```bash
python acknowledgment.py
```

Refer to the instructions under [`Sequences`](sequences) for using custom sequences as input.
